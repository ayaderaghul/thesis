Introduction

I simulate populations of agents playing games for fitness and the population evolves under an evolutionary algorithm. The Racket code is on my Github page, further information is on the Gitlab page of this project. The general sentiment is that I put the project in the freest form possible in accordance with local regulation. In the best case, you would get it in public domain under CC0.

First of all, a player is a distinct set of interest.

Each player can choose a strategy from a strategy set and the outcome would be combined in a matrix with corresponding payoffs

Here is the payoff matrix of a very popular game: the Prisoner's Dilemma. In this game, there are 2 players and each has 2 possible strategies. Therefore there can be 4 possible outcomes. We would proceed to reason about this situation in the next section.
